HydroShare Time Series Vis

This page lists links to web-based, interactive graphs, plots, and visualizations of sample data from the HydroShare project. The purpose of this effort is to explore visual representations and interactive analytical tools to support exploration and analysis of time series data.

Currently individual charts are shown but they can be easily combined into a panel showing multiple charts. Standard data visualization techniques such as linking (connecting user actions across multiple charts) and brushing (selecting a set of data points in linked charts) will be explored to provide analytical capabilities.
